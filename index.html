Methodology

Walk-forward backtesting.
The honest way to test.

Traditional backtests often cheat — they train on future data. Alpha Signal uses walk-forward validation: models see only what a real investor would have known at each point in time.

// 01
📊
10+ Data Sources
Yahoo Finance, CoinGecko, Binance, FRED, NewsAPI, Reddit, and Ollama LLM sentiment. 76 engineered features per signal — price, volume, volatility, momentum, macro, and cross-asset.
// 02
🔄
16 Quarterly Windows
Q1 2022 through Q4 2025. Each window trains on all prior data and tests on the next quarter with an expanding window — models retrained from scratch each fold. No future data leakage.
// 03
🤖
Autonomous Agent Loop
Agent Alpha runs hourly: observe signals → reason with regime-aware rules → decide positions → execute paper trades → reflect weekly. If SHORT accuracy drops below 45%, it proposes a retrain.
// Walk-Forward Expanding Window Validation
Platform Architecture

Four layers.
One cohesive platform.

Built entirely in Rust, running 24/7 on a home NUC in London. Every layer talks to the next via a typed REST API. No managed services. No cloud dependencies in the critical path.

The Ensemble

Six models.
One decision.

No single model has an edge in all market regimes. The ensemble combines classical statistics, gradient boosting, and deep learning — each votes, confidence-weighted. Signal: BUY if ensemble P(up) > 0.55 and walk-forward accuracy > 51%.

// Classical · Gradient Descent
Linear Regression
Baseline return predictor. 3,000 epochs, 10 features. Captures linear factor exposures: momentum, reversion, volume trends. Interpretable weights.
Interpretable
// Classical · Sigmoid Loss
Logistic Regression
Binary UP/DOWN classifier. 3,000 epochs. Fast, regularised, provides calibrated probability estimates. The ensemble's probabilistic anchor.
Probability-calibrated
// Boosting · Custom Rust CART
Gradient Boosted Trees
Custom Rust implementation — CART trees with boosting, no external library. Captures non-linear feature interactions. Robust to outliers in trending markets.
Non-linear
// Boosting · C++ Bindings
LightGBM
Leaf-wise tree growth with histogram binning. Faster than XGBoost with comparable accuracy. Used as the second boosting vote in the ensemble.
High-speed
// Deep Learning · Candle-NN
LSTM + GRU
Long Short-Term Memory and Gated Recurrent Unit — learn sequential patterns across price history. hidden_size=64, seq_length=10–20. Captures regime memory.
Temporal patterns
// Meta-Model · Stacking
Market Regime Classifier
Detects BULL/NEUTRAL/BEAR/CRISIS from SPY returns, VIX, and risk_score. Adjusts position scaling (1.0x → 0.18x) and suspends SHORTs if accuracy < 50%.
Adaptive

These results come from a walk-forward backtest — 16 quarterly windows where models were trained strictly on past data before each test period. This is how practitioners in quantitative finance validate strategies; traditional backtests that train on the full dataset are not credible.

The ensemble outperformed SPY across a period that included the 2022 bear market, the 2023 tech recovery, and 2024–25 AI-driven growth. It performed best as drawdown protection — reducing losses in falling markets while capturing upside in rising ones.

Live accuracy sits at 56.4% (BUY: 56.2%, SHORT: 100%) across 858 resolved signals with a profit factor of 1.28. The agent is now tracking 3,703 paper trades across stocks, FX, and crypto — 1,421 closed trades across 29 pages of history.
